# โปรดอ่าน
## ⚙️ วิธีติดตั้งและรันโปรเจกต์

### 1. ดาวน์โหลดเเละเเตกไฟล์เอาโฟล์เดอร์L_13ที่อยู่ข้างในออกมาเเล้วเปิดโฟล์เดอร์ที่เอาออกมาไปเปิดในVscodเเล้วใช้คำสั่งนี้ในTerminal
```bash
cd L_13
```

### 2. สร้าง venv ขึ้นมาใหม่(ไม่งั้นจะเเสดงไม่ได้)
```bash
python -m venv venv
```

### 3. ใชhคำสั่งนี้เพื่อเข้า venv
```bash
venv\Scripts\activate
```

### 4. หลังจากมี venv อยู่นำหน้าชื่อตรงTerminalให้ติดตั้ง django ลงvenv
```bash
pip install django

```

### 5. เข้าโฟลเดอร์โปรเจกต์
```bash
cd myproject
```

### 6. รันเซิร์ฟเวอร์
```bash
python manage.py runserver
```

### 7. เปิดเบราว์เซอร์
```
http://127.0.0.1:8000/
```
